PSICOLOGIA DAS CORES

Reprodução de cores por definição

>> NOME:
style{background-color:blue;color:white;}
>> CÓDIGO HEXADECIMAL:
style{background-color:#0000ff; color:#ffffff;}
>> CÓDIGO RGB (RED, GREEN, BLUE):
style{background-color:rgb(0, 0, 255); color:rgb(255, 255, 255);}
>> CÓDIGO HSL (HUE, SATURATION, LIGHTNESS):
style{background-color:hsl(240, 100%, 50%); color:hsl(0, 0%, 100%);}

Harmonia de cores

Importante estudar sobre Psicologia das cores para definição de paletas e sua harmonização.



Ferramentas auxiliares

Segue abaixo recomendações de ferramentas para definição de paleta de cores:

  • Adobe Colors
  • Paletton
  • Coolors

Vale destacar também que temos uma ferramenta poderosa para nos auxiliar no próprio navegador, por meio da extensão Colorzilla, que nos permite capturar a configuração de cores, gerando um código Hexadecimal, RGB ou HSL de uma página web, pdf, ou demais que estejam sendo apresentados na tela.
